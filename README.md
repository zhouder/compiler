# compiler

一个面向**基本 C 子集**的简易编译器课程设计项目。编译流程包含词法分析、语法分析、语义分析、中间代码生成和目标代码生成，目标代码为教学用伪汇编。

## 编译流程

| 阶段 | 输入 | 输出 | 主要文件 |
|---|---|---|---|
| 词法分析 | C 子集源程序字符串 | Token 序列，包含行号、列号、类别、词素 | `src/lexer/` |
| 语法分析 | Token 序列 | AST 抽象语法树 | `src/parser/` |
| 语义分析 | AST | 符号表检查结果，发现重复定义、未定义变量、类型不匹配等错误 | `src/semantic/` |
| 中间代码生成 | AST | 四元式序列 | `src/ir/` |
| 目标代码生成 | 四元式序列 | 教学用伪汇编代码 | `src/codegen/` |

## 支持的语言子集

词法层面支持：

- 关键字、标识符
- 十进制整数、八进制整数、十六进制整数
- 浮点数
- 字符常量、字符串常量
- 运算符和界符
- 单行注释 `//`、多行注释 `/* */`
- `#include <stdio.h>` 这类预处理行的识别和跳过

语法和语义层面支持：

- 类型：`int`、`char`、`float`、`void`
- 函数：无参函数定义，重点支持 `main`
- 变量定义：`int a;`、`float x;`、`char ch;`
- 变量初始化：`int a = 1;`
- 赋值语句：`a = expr;`
- 顺序语句
- 复合语句块：`{ ... }`
- 选择语句：`if`、`if / else`、嵌套 `if`
- 循环语句：`while`、`for`
- 返回语句：`return expr;`
- 输出语句：`printf(...)`
- 输入语句：`scanf("%d", &a);`
- 算术表达式：`+`、`-`、`*`、`/`、`%`
- 关系表达式：`<`、`<=`、`>`、`>=`、`==`、`!=`
- 逻辑表达式：`&&`、`||`、`!`
- 一元表达式：`+`、`-`、`!`、`&变量`
- 括号表达式：`(expr)`

语义分析支持：

- 作用域符号表
- 重复定义检查
- 未定义变量检查
- 基础类型检查
- `char -> int`、`int/char -> float` 的赋值兼容
- `scanf` 输入参数必须为 `&变量`

## 暂不支持

- 有参函数定义和完整函数调用检查
- 数组、指针运算、结构体、枚举
- 多变量声明，例如 `int a, b;`
- 自增自减和复合赋值，例如 `i++`、`a += 1`
- `break`、`continue`、`do while`、`switch case`
- 真实可汇编运行的 x86/MIPS 汇编输出

## 运行

```bash
python src/main.py examples/test1.c
```

运行后会在控制台打印以下内容：

- `TOKENS`
- `AST`
- `SEMANTIC`
- `IR`
- `ASM`

同时会把每个阶段结果保存到 `output/`：

- `output/test1.tokens.txt`
- `output/test1.ast.txt`
- `output/test1.ir.txt`
- `output/test1.asm`
- `output/test1.log.txt`
