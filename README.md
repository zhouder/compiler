# compiler

一个面向**基本 C 子集**的简易编译器课程设计项目。

当前版本包含：

- 词法分析
- 语法分析（递归下降）
- 语义分析（符号表、重复定义、未定义变量、基础类型检查）
- 中间代码生成（四元式）
- 目标代码生成（教学用伪汇编）

## 支持的语言子集

- 类型：`int` `char` `float` `void`
- 函数：支持无参函数定义，重点支持 `main`
- 语句：
  - 变量定义
  - 赋值语句
  - 表达式语句
  - `if / else`
  - `while`
  - `for`
  - `return`
  - `printf`
  - `scanf`
- 表达式：
  - `+ - * / %`
  - `== != < <= > >=`
  - 一元 `+ - !`
  - 括号

## 运行

```bash
python src/main.py examples/test1.c
```
